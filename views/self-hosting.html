Minimal Requirements

ProfitLocks is not hard, but it requires you to be able to type in commands into a text based command window, move files around, download and run new programs on your computer, edit files in a basic text editor, and do other things that are not “graphical” in nature. If you are a “Point and Click” type of person, you might want to look for some help with installation before you forge headlong into this adventure.

That said, we have a great support team, and can help you through the installation and setup if you are patient and are willing to learn some new skills.

You:

  • Know what a file extension is, and how to enable these in Windows
  • Know how to unzip a file and know where to find the unzipped files (this is tricky in Windows!)
  • Know what a cmd prompt or terminal window is, and at least can navigate to a directory in it
  • Know how to edit *.properties files and that a '#' is not a Twitter hashtag
  • Know that running this on your MacBook Pro is like doing grocery shopping with your Lambo
  • Know how to read and understand technical writing

Windows:

  • Windows 7 or higher (32/64 bit)
  • Min. 2GB ram
  • Min. 500MB free space
  • A decent text editor like Notepad++ https://notepad-plus-plus.org/

MacOS:

  • Have you ever used the “Terminal” app on your Mac before? Do you know what Unix commands are?
  • If your can answer yes. Please use the Mac guide here.

Linux:

  • Min. 2GB ram
  • Min. 500MB free space

Raspberry Pi:

  • Raspberry Pi 3 or newer
  • Min. 8GB MicroSD card

Internet connection:

  • Min. 5 Mbps per instance
  • Min. 1TB a month data cap. (ProfitLocks can use about 30GB a day)
