Telegram Chat Bot

Setup Telegram bot to Config your chat bot with Trading Updates. This user guide allow you from start to fly.

In Telegram, send a private message to @BotFather (NOT to be confused with TheBotFather ​channel)

  • Once there type “/​newbot” ​to create a bot. Answer the questions ​by typing answers and pressing enter.
  • You will receive a message ​containing your bot token.

Start a chat with your bot's name (for example ​@MyTPBot) and press start button ​

Start another ​chat with https://t.me/MyTelegramID_bot.

  • If that doesn't work, try a chat with https://​telegram.me/​id_chatbot
  • or paste @id_chatbot into a telegram chat and click it. It should open the chat id bot in a new chat.
  • or send a message to: @userinfobot with content: /start
  • Finally, here is a link to a page https://ebotstore.de/bot/mytelegramidbot-telegram with a couple more ID chatbots for telegram that you can click if the above doesn't get a valid response.

Edit application.properties ​and set : 

TELEGRAM_BOT_TOKEN = your bot token (ex: 125634789:AdGsJXvOnWxasdtasrfanmvlQQfb-aaa)
TELEGRAM_CHAT_ID = your chat ID (ex: 1234567890 or -1234567890 for groups)

Stop and restart ProfitLocks. Remember if in Mac or Linux to CTRL-C the process if in a terminal window or stop the pm2 process or other process manager.

Go to http://localhost:8081/settings/telegramTestMessage to test your telegram setup. You will receive a message ​if the setup is correct.

NOTES:

  1. You will not receive any messages when in testmode.
  2. For group chats invite https://​telegram.me/​myidbot to the existing group and type /getgroupid@myidbot (include the '-' symbol when adding it to your application.properties file
  3. Remember to click “Start” at the bottom of the page when you start a new chat.
